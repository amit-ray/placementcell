Project : Placement-cell

Environment   : Java Servlets, MySQL, HTML, JavaScript, CSS

Description	: In this project I have developed a web application of placement cell using Tomcat Apache 9  server.
The portal has admin and student login features. Admin can add list of companies and criteria for job application.
New students can register through the web page.
After logged in to the system students can see the job posts, company details, eligibility criteria and interview date etc.
along with other details.
